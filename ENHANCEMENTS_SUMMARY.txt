HOW TO USE FOR FILE SEARCHING
============================================================

Example Workflow:
  1. "open explorer"      → Opens File Explorer
  2. "down down down"     → Navigate down 3 folders
  3. "enter"              → Open selected folder
  4. "up"                 → Go back up one level
  5. "home"               → Jump to start
  6. "down"               → Move to first item
  7. "enter"              → Open file/folder
  8. "left"               → Go to parent folder
  9. "right"              → Go into folder
  10. "end"               → Jump to last item
